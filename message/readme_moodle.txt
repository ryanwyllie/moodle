EmojiOne Area
-------------

Instructions to import 'EmojiOneArea' into Moodle:

1. Download the latest release from https://github.com/mervick/emojionearea/tree/master/dist/emojionearea.js
2. Copy into message/amd/src/emojionearea.js
3. Run grunt to build the JS
4. Download the latest release from https://github.com/mervick/emojionearea/tree/master/dist/emojionearea.min.css
5. Copy into message/emojionearea.min.css
